q. what is key in react?

-> key is a special attribute used when rendering lists of elements in react.

-> {items.map(item => (
    <li key={key.id}>{item.name}></li>
))}


why key is important?

keys help react identify which items have changed, been added or removed.

react uses keys durig reconciliation (diffing process) to update the ui efficiently .



key is a unique identifier used in react lists to help react rrack changes in elements. it imporoves performance by allowing react to update only the chanegs items instead of re-rendering the entire list.



without keus , react cannot efficiently track element chanegs whihc may lead to unnecesary re-renders.
